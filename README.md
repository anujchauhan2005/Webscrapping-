 # 🛒 Amazon Laptop Price Web Scraper

## 📌 Project Overview
This project is a **web scraping application** that extracts laptop prices from Amazon and stores the data in a CSV file for further analysis.

---

## 🚀 Features
- 🔍 Scrapes laptop names and prices from Amazon
- 📊 Stores data in structured CSV format
- ⚡ Fast and automated data collection
- 🧹 Clean and readable output

---

## 🛠️ Technologies Used
- Python 🐍
- BeautifulSoup (for parsing HTML)
- Requests (for sending HTTP requests)
- Pandas (for saving data into CSV)
